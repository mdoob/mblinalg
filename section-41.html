Section 6.3 First properties of linear transformations

A linear transformation \(L\colon\R^m\to\R^n\text{,}\) by definition, satisfies

  • \(L(\vec x+\vec y)=L(\vec x)+L(\vec y)\) and

  • \(\displaystyle L(r\vec x)=rL(\vec x)\)

Theorem 6.3.1. (First properties).

Any linear transformation \(L\) satisfies

  • \(\displaystyle L(\vec0)=\vec0\)

  • \(\displaystyle L(\vec x-\vec y)=L(\vec x)-L(\vec y)\)

  • \(L(\vec x-\vec y)=\vec0\) if and only if \(L(\vec x) =L(\vec y)\)

  • \(\displaystyle L(r_1\vec x_1+r_2\vec x_2)=r_1L(\vec x_1)+r_2L(\vec x_2)\)

  • For any vectors \(\vec x_1, \vec x_2,\ldots,\vec x_n\) and real numbers \(r_1, r_2,\ldots,r_n\)

    \begin{equation*} L(r_1\vec x_1+r_2 \vec x_2+\cdots+r_n\vec x_n)= r_1L(\vec x_1)+r_2 L(\vec x_2)+\cdots+r_nL(\vec x_n) \end{equation*}
Proof.

  1. We evaluate \(L(\vec 0+\vec 0)\) in two ways:

    1. Since \(\vec 0+\vec 0=\vec0\text{,}\) we have \(L(\vec 0+\vec 0)=L(\vec 0)\)

    2. Since \(\vec 0+\vec 0=2\vec0\text{,}\) we have \(L(\vec 0+\vec 0)= L(2\vec 0)=2L(\vec 0)\text{,}\) and so

      \begin{equation*} 2L(\vec 0)=L(\vec 0)\\L(\vec 0)=\vec0 \end{equation*}

  2. \(\displaystyle L(\vec x-\vec y) =L(\vec x+(-1)\vec y) =L(\vec x)+L((-1)\vec y) L(\vec x)-L(\vec y)\)

  3. \(L(\vec x-\vec y)=\vec0\) if and only if \(L(\vec x)-L(\vec y)=\vec0\) which in turn implies \(L(\vec x) =L(\vec y)\text{.}\)

  4. \(\displaystyle L(r_1\vec x_1+r_2\vec x_2) =L(r_1\vec x_1)+L(r_2\vec x_2) =r_1L(\vec x_1)+r_2L(\vec x_2) \)

  5. We apply the addition property repeatedly:

    \begin{equation*} \begin{array}{rl} L(r_1\vec x_1\amp+r_2 \vec x_2+\cdots+r_n\vec x_n)\\ \amp = L(r_1\vec x_1)+L(r_2 \vec x_2+\cdots+r_n\vec x_n)\\ \amp = r_1L(\vec x_1)+ L(r_2\vec x_2+\cdots+r_n\vec x_n)\\ \amp = r_1L(\vec x_1)+ L(r_2\vec x_2)+L(r_3\vec x_3+\cdots+r_n\vec x_n)\\ \amp = r_1L(\vec x_1)+ r_2L(\vec x_2)+L(r_3\vec x_3+\cdots+r_n\vec x_n)\\ \amp \,\,\vdots\\ \amp = r_1L(\vec x_1)+r_2 L(\vec x_2)+\cdots+r_nL(\vec x_n) \end{array} \end{equation*}

The standard basis of \(\R^m\) is the set of vectors \(\{\vec e_1, \vec e_2,\ldots,\vec e_m\}\) where

\begin{equation*} \vec e_1=(1,0,\ldots,0)\\ \vec e_2=(0,1,\ldots,0)\\ \vdots\\ \vec e_m=(0,0,\ldots,1) \end{equation*}
Theorem 6.3.2. (The value of \(L\) on the standard basis determines \(L\) everywhere).

If the values \(L(\vec e_1), L(\vec e_2),\ldots L(\vec e_m)\) are known, then value of \(L(\vec x)\) is known for all \(\vec x\) in \(\R^m\text{.}\)

Proof.

Suppose \(L(\vec e_1)=\vec f_1, L(\vec e_2)=\vec f_2,\ldots,L(\vec e_m)=\vec f_m\text{,}\) and \(\vec x=(x_1,x_2,\ldots,x_m)\text{.}\) Then \((x_1,x_2,\ldots,x_m)=x_1\vec e_1 + x_2\vec e_2+\cdots+x_m\vec e_m\) and

\begin{equation*} \begin{array}{rl} L(\vec x) \amp =L(x_1\vec e_1 + x_2\vec e_2+\cdots+x_m\vec e_m)\\ \amp =x_1L(\vec e_1) + x_2L(\vec e_2)+\cdots+x_mL(\vec e_m)\\ \amp =x_1\vec f_1 + x_2\vec f_2+\cdots+x_n\vec f_m \end{array} \end{equation*}
Corollary 6.3.3. \(L(\vec e_i)=\vec 0, i=1\cdots m\text{,}\) implies \(L=0\).

If \(L(\vec e_1)=L(\vec e_2)=\cdots=L(\vec e_m)=\vec 0\text{,}\) then \(L\) is the zero transformation.

There is an easy consequence:

Theorem 6.3.4. Two linear transformations equal on the standard basis are equal everywhere.

Suppose \(L_1\colon \R^m\to\R^n\text{,}\) \(L_2\colon \R^m\to\R^n\) and \(L_1(\vec e_i) = L_2(\vec e_i)\) for \(i=1,2,\dots m\text{.}\) Then \(L_1(\vec x)=L_2(\vec x)\) for all \(\vec x\) in \(\R^m\text{,}\) and \(L_1=L_2\text{.}\)

Proof.

\(L_1(\vec e_i)-L_2(\vec e_i)=\vec 0\) by assumption. If \(\vec x=(x_1,\dots,x_m)\text{,}\) we may write \(\vec x = x_1\vec e_1+x_2\vec e_2+\cdots +x_m\vec e_m.\) Then

\begin{align*} L_1(\vec x) \amp =L_1(x_1\vec e_1+x_2\vec e_2+\cdots +x_m\vec e_m)\\ \amp =x_1L_1(\vec e_1)+x_2L_1(\vec e_2)+\cdots +x_mL_1(\vec e_m)\\ \amp =x_1L_2(\vec e_1)+x_2L_2(\vec e_2)+\cdots +x_mL_2(\vec e_m)\\ \amp =L_2(x_1\vec e_1+x_2\vec e_2+\cdots +x_m\vec e_m)\\ \amp= L_2(\vec x) \end{align*}

From this we see that

\begin{equation*} L_1(\vec x)-L_2(\vec x) = \vec 0. \end{equation*}
