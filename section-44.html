Section 6.6 Matrix representation of a transformation

Suppose we have a linear transformation \(L\colon \R^m\to\R^n\text{.}\) As usual, let \(\{\vec e_1,\vec e_2,\ldots,\vec e_m\}\) be the standard basis for \(\R^m\text{.}\) Consider the vectors \(\{L(\vec e_1) ,L(\vec e_2), \ldots, L(\vec e_m)\}\) in \(\R^n\text{.}\) We form a matrix \(A\) by using the entries of \(L(\vec e_k)\) for the \(k\)-th column. We can think of the construction in the following way:

\begin{equation*} \begin{array}{cccc} A = \amp [L(\vec e_1) \amp L(\vec e_2) \amp \cdots \amp L(\vec e_m)]\\ \amp \uparrow \amp \uparrow \amp \amp \uparrow\\ \amp \text{column } 1\amp \text{column } 2\amp\amp \text{column } m \end{array} \end{equation*}

Notice that \(A\) is an \(n\times m\) matrix, and hence we have \(L_A\colon\R^m\to\R^n\text{.}\) We evaluate this transformation at \(\vec e_1\text{:}\)

\begin{equation} L_A(\vec e_1)= A\begin{bmatrix} 1\\0\\\vdots\\0 \end{bmatrix} =L(\vec e_1)\label{MatrixRep}\tag{6.6.1} \end{equation}

Similarly

\begin{equation*} L_A(\vec e_k)=L(\vec e_k) \text{ for } 1\leq k\leq n \end{equation*}

As we have seen in TheoremÂ 6.3.2 a linear transformation is determined by its values on the standard basis. This implies \(L=L_A\text{.}\) We call the constructed matrix \(A\) the matrix representation of \(L\text{.}\)

Theorem 6.6.1. (Matrix representation of a linear transformation).

Let \(L\colon \R^m\to\R^n\) be a linear transformation, and let \(A\) be the matrix formed by letting the \(k\)-th column be the entries of \(L(\vec e_k)\text{.}\) Then

\begin{equation*} L=L_A \end{equation*}
Proof.

We can compute this in a different way: The equation (6.6.1) says

\begin{align*} L(\vec e_1) \amp=A\begin{bmatrix}1\\0\\\vdots\\0\end{bmatrix}\\ \amp=\begin{bmatrix}a_{1,1}\\a_{2,1}\\\vdots\\a_{n,1}\end{bmatrix}\\ \amp=a_{1,1}\vec e_1+a_{2,1}\vec e_2+\cdots+a_{n,1}\vec e_n \end{align*}

and, similarly,

\begin{equation*} L(\vec e_i)=a_{1,i}\vec e_1+a_{2,i}\vec e_2+\cdots+a_{n,i}\vec e_n \end{equation*}
