Suppose that \(A\) is an \(n\times n\) square matrix. Then the following statements are equivalent:

  1. \(A\) is invertible

  2. \(A\mathbf x=0\) if and only if \(\mathbf x=0\)

  3. The reduced row echelon form of \(A\) is \(I_n\)

  4. \(A\) is a product of elementary matrices

  5. \(A\mathbf x=\mathbf b\) is consistent for any \(\mathbf b\)

  6. \(A\mathbf x=\mathbf b\) has exactly one solution for any \(\mathbf b\)

  7. \(\displaystyle \det A\not=0\)
