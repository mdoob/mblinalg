Theorem 4.6.6. Different equations for the same plane.

The two equations

\begin{equation*} a_1x+b_1y+c_1z+d_1=0\\ a_2x+b_2y+c_2z+d_2=0 \end{equation*}

are equations of the same plane if and only if

\begin{equation*} (a_2,b_2,c_2,d_2)=r (a_1,b_1,c_1,d_1) \textrm{ for some } r\not=0. \end{equation*}
