This takes a little trick. Start with the permutation

\begin{equation*} \sigma= \begin{pmatrix} 1\amp 2\amp 3\amp \cdots\amp n\\ i_1\amp i_2\amp i_3\amp \cdots\amp i_n \end{pmatrix} \end{equation*}

and rearrange the columns so that the bottom row is \(123\cdots n\text{.}\) For example, if \(\sigma= \begin{pmatrix} 123456\\324165\end{pmatrix}\text{,}\) rearranging the colums gives the permutation \(\begin{pmatrix} 421365\\123456\end{pmatrix}\text{.}\)
