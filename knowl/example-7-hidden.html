Consider the following system of linear equations:

\begin{equation*} \begin{array}{rl} x_1-x_2+2x_3-x_4 \amp = -1 \\ 2x_1+x_2-2x_3-2x_4 \amp = -2 \\ -x_1+2x_2-4x_3+x_4 \amp = 1 \\ 3x_1-3x_4 \amp = -3 \end{array} \end{equation*}

The augmented matrix is then put into reduced row echelon form:

\begin{align*} \begin{bmatrix} 1 \amp -1 \amp 2 \amp -1 \amp -1\\ 2 \amp 1 \amp -2 \amp -2 \amp -2\\ -1 \amp 2 \amp -4 \amp 1 \amp 1\\ 3 \amp 0 \amp 0 \amp -3 \amp -3 \end{bmatrix} \amp\amp\amp \begin{array}{rl} R_2 \amp \gets R_2-2R_1\\ R_3 \amp \gets R_3+R_1\\ R_4 \amp \gets R_4-3R_1 \end{array}\\\\ \begin{bmatrix} 1 \amp -1 \amp 2 \amp -1 \amp -1\\ 0 \amp 3 \amp -6 \amp 0 \amp 0\\ 0 \amp 1 \amp -2 \amp 0 \amp 0\\ 0 \amp 3 \amp -6 \amp 0 \amp 0 \end{bmatrix} \amp\amp\amp \begin{array}{rl} R_2 \amp \gets \tfrac13 R_2\\ \end{array}\\\\ \begin{bmatrix} 1 \amp -1 \amp 2 \amp -1 \amp -1\\ 0 \amp 1 \amp -2 \amp 0 \amp 0\\ 0 \amp 1 \amp -2 \amp 0 \amp 0\\ 0 \amp 3 \amp -6 \amp 0 \amp 0 \end{bmatrix} \amp\amp\amp \begin{array}{rl} R_1 \amp \gets R_1 + R_2\\ R_3 \amp \gets R_3- R_2\\ R_4 \amp \gets R_4- 3R_2 \end{array}\\\\ \begin{bmatrix} {\color{red} 1} \amp 0 \amp 0 \amp -1 \amp -1\\ 0 \amp {\color{red} 1} \amp -2 \amp 0 \amp 0\\ 0 \amp 0 \amp 0 \amp 0 \amp 0\\ 0 \amp 0 \amp 0 \amp 0 \amp 0 \end{bmatrix} \end{align*}

Since \(x_3\) and \(x_4\) are free variables, we assign them parameters: \(x_3=s\) and \(x_4=t\text{.}\) We can now evaluate the constrained variables: \(x_1=-1+t\) (from the first row) and \(x_2=2s\) (from the second row). In short, \((x_1,x_2,x_3,x_4)=(-1+t,2s,s,t)\) for any choice of \(s\) or \(t\text{.}\)

Now, as a check, let's put these solutions back into the original equations.

\begin{equation*} \begin{array}{rlll} x_1-x_2+2x_3-x_4 \amp =\amp (-1+t) - (2s) + 2(s) - (t) \amp= -1\\ 2x_1+x_2-2x_3-2x_4 \amp =\amp 2(-1+t) + 2s -2s-2t \amp= -2\\ -x_1+2x_2-4x_3+x_4 \amp =\amp -(-1+t) +2(2s) -4s+t \amp= 1 \\ 3x_1-3x_4 \amp =\amp 3(-1+t) -3t \amp= -3 \end{array} \end{equation*}
