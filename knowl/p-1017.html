If the set \(\mathcal{X}=\{\vec x_1,\vec x_2,\ldots,\vec x_m\}\) is a linearly independent and \(\vec w\) is a linear combination of vectors in \(\mathcal{X}\text{,}\) then

\begin{equation} \vec w=r_1\vec x_1+r_2\vec x_2+\cdots+r_m\vec x_m \text{ uniquely}\text{.}\tag{4.9.1} \end{equation}

By uniquely we mean that there is exactly one choice of \(r_1,r_2,\ldots,r_m\) so that (4.9.1) is valid.
