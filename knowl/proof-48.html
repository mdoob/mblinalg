Proof

We proceed by considering three cases:

  1. \(\det B=0\text{:}\) In this case \(\det A\;\det B=0\text{.}\) In addition, from Theorem 3.4.21, The is an \(\mathbf x\not=0\) so that \(B\mathbf x=0\text{.}\) Then \(AB\mathbf x=0\) and so \(\det(AB)=0\text{.}\) This gives

    \begin{equation*} \det(AB)=0=\det A\;\det B. \end{equation*}

  2. \(\det B\not=0\) and \(\det A=0\text{:}\) Once again, \(\det A \det B=0\text{.}\) Again, using Theorem 3.4.21 (twice), there is \(\mathbf y\not=0\) so that \(A\mathbf y=0\) and there is an \(\mathbf x\) so that \(B\mathbf x=\mathbf y\text{.}\) Notice that \(\mathbf x\not=0\text{,}\) for otherwise \(\mathbf y=B\mathbf x=0\text{.}\) We then have \(AB\mathbf x=A\mathbf y=0\) with \(\mathbf y\not=0\text{,}\) and so \(\det(AB)=0\text{,}\) which, once again gives

    \begin{equation*} \det(AB)=0=\det A\;\det B. \end{equation*}

  3. \(\det B\not=0\) and \(\det A\not=0\text{:}\) Once again, using Theorem 3.4.21,

    \begin{equation*} A=F_1 F_2\cdots F_k \textrm{, a product of elementary matrices.} \end{equation*}

    Using Theorem 3.4.16 repeatedly,

    \begin{equation*} \det A= \det F_1\;\det F_2\cdots\det F_k \end{equation*}

    and so

    \begin{equation*} \det A\; \det B=\det F_1\;\det F_2\cdots\det F_k \det B \end{equation*}

    But also by using Theorem 3.4.16 repeatedly,

    \begin{align*} \det(AB) \amp =\det(F_1F_2\cdots F_k B)\\ \amp =\det F_1\;\det F_2\cdots\det F_k\;\det B\\ \amp =\det A\;\det B \end{align*}
